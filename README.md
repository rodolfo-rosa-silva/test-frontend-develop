# test-frontend

FRONTEND (REACTJS)

## How to

1. Install eslint & prettier pluggins in VS Code
2. Activate pretties as a default formatter
3. Run commands:

## javascript local
```
yarn install
yarn start:dev
```
## javascript build:dev
```
yarn install
yarn build:dev
```

## docker build:dev and run
```
docker build -t test-frontend .
docker run -p 8080 -t test-frontend
```
